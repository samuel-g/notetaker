
## Getting Started

### Prerequisites

- **JDK 17 or later**
- **Gradle 7.0 or later**

### Installation

**Clone the repository:**
   ```
   git clone https://github.com/samuel-g/notetaker.git
   cd notetaker
   ```

**Build the project:**
```./gradlew build```

**Run the application:**
```./gradlew bootRun```
The application will start and be accessible at http://localhost:8080.

**API Endpoints**
* GET /api/notes
* GET /api/notes/{id}
* POST /api/notes
* PUT /api/notes/{id}
* DELETE /api/notes/{id}
